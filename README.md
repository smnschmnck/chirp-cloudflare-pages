# Chirp

Twitter clone hosted on Cloudflare Pages using Pages Functions as the backend and Cloudflare D1 as the primary database.

[Demo](https://cf-react-2k0.pages.dev/)

Tech used:
 - React
 - TypeScript
 - tRPC
 - Tailwind
 - Lucia Auth
 - Drizzle ORM
 - TanStack Router
 - TanStack React Query
 - Vite
